# NextJS App Using Apollo Client & AG Grid

Instructions:

- Clone this repo
- Run:

```
yarn install
yarn dev
```

- Goto

```
http://localhost:3000/
```

Built using my boiler plate with:

- NextJS
- Typescript
- React Testing Library
- SASS
- AG Grid React
- Prettier
- ESLint
- Bootstrap
- Material UI
- Theming using Styled Components
- GraphQL (via ApolloClient)
- Modal Provider
- more...
